# Titel: Als User möchte ich eine Neue Notiz anlegen und speichern können

## Inhalt
Als User möchte ich die Möglichkeit haben, eine neue Notiz zu erstellen und diese zu speichern. Dies umfasst das Eingeben von Text, das Speichern der Notiz und das Bestätigen, dass die Notiz erfolgreich gespeichert wurde.

## Akzeptanzkriterien
- Der User kann eine neue Notiz erstellen.
- Der User kann Text in die Notiz eingeben.
- Der User kann die Notiz speichern.
- Nach dem Speichern wird eine Bestätigung angezeigt, dass die Notiz erfolgreich gespeichert wurde.
- Die gespeicherte Notiz kann später wieder abgerufen werden.
